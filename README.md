## Python Moving Average Example Code using Pandas

### Prerequsite

* make
* virtuanenv
* numpy
* pandas
* matplotlib

```
sudo apt update 
sudo apt install make
```

Create Python Virtual Environment
```
virtualenv venv
. venv/vin/activate
pip install -r requirements.txt
```

### Generate Charts

Generate individual chart
```
python weight.py
python sma.py
python cma.py
python wma.py
python ema.py
```


Generate whole charts
```
make
```
